# 서울시 자전거 실시간 정보 시스템

## 프로젝트 설명

서울시의 공공 자전거 대여소 정보를 실시간으로 확인할 수 있는 웹 애플리케이션입니다. 카카오 맵을 이용하여 서울시 전역의 자전거 대여소 위치와 자전거 가용 현황을 시각적으로 표시합니다.

## 주요 기능

### 1. **실시간 대여소 정보 조회**

- 서울시의 모든 자전거 대여소 정보를 API를 통해 실시간으로 로드
- 페이지 단위 로드로 대규모 데이터셋 효율적 처리 (페이지 크기: 1,000개)

### 2. **지도 기반 시각화**

- 카카오 맵 API를 활용한 대여소 위치 표시
- 각 마커에 남은 자전거 수를 시각적으로 표현:
  - 🔴 빨강: 자전거 없음 (0대)
  - 🟡 노랑: 자전거 부족 (1~4대)
  - 🟢 초록: 자전거 풍부 (5대 이상)

### 3. **마커 클러스터링**

- 지도 축소 시(줌 레벨 8 이상) 자동으로 근처 마커를 클러스터링
- 지도 확대 시 개별 마커 표시
- 부드러운 사용자 경험 제공

### 4. **사용자 위치 기반 서비스**

- 사용자의 현재 위치를 자동으로 감지 (지오로케이션)
- 현재 위치를 지도에 표시
- 현재 위치 기준으로 지도 자동 중심 설정

### 5. **상세 정보 조회**

- 마커 클릭 시 대여소의 상세 정보 팝업 표시:
  - 대여소 명
  - 거치대 개수
  - 남은 자전거 개수
  - 거치율(%)

### 6. **지오코딩**

- 좌표 정보가 없는 대여소의 주소로부터 좌표 자동 변환
- 서울시 공공 API의 일관되지 않은 데이터 형식에 대응

## 기술 스택

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **지도 API**: 카카오 맵 JavaScript SDK
- **데이터 API**: 서울시 공공데이터 포털 (자전거 대여현황)
- **위치 서비스**: HTML5 Geolocation API

## 파일 구조

```
자전거.html          # 메인 애플리케이션 파일 (HTML, CSS, JavaScript 포함)

```

## 사용 방법

1. 브라우저에서 `자전거.html` 파일을 열기
2. 지도가 로드되고 자동으로 데이터 로드 시작
3. 마커를 클릭하여 대여소 상세 정보 확인
4. 지도 확대/축소로 원하는 지역의 자전거 현황 확인

## API 정보

### 서울시 자전거 대여소 정보 API

- **출처**: 서울시 공공데이터 포털
- **엔드포인트**: `http://openapi.seoul.go.kr:8088/[SERVICE_KEY]/json/bikeList/[START]/[END]/`
- **페이지 크기**: 1,000개 단위
- **응답 형식**: JSON

### 카카오 맵 API

- **서비스**: 지도 표시, 마커 관리, 마커 클러스터링, 지오코딩
- **라이브러리**: services, clusterer, drawing

## 주요 기술 특징

### 1. **효율적인 데이터 로드**

- 페이지 단위로 데이터 로드하여 메모리 효율성 확보
- 비동기 처리로 UI 블로킹 방지

### 2. **동적 마커 관리**

- SVG 기반 커스텀 마커로 자전거 수를 시각적으로 표현
- 마커 색상 동적 변경으로 상태 표시

### 3. **줌 레벨 기반 클러스터링**

- 줌 레벨 변경 시 자동으로 클러스터 상태 업데이트
- 사용자 경험 최적화

### 4. **안정적인 지오코딩**

- 서울 공공 API의 불일치하는 데이터 형식 처리
- 주소 전처리로 지오코딩 성공률 향상

## 참고사항

- 지오로케이션 기능은 HTTPS 또는 localhost 환경에서 정상 작동
- 자전거 API 서비스 키는 공개 키를 사용 중 (변경 가능)
- 마커 클러스터링은 대규모 데이터셋 표시 시 성능 최적화
